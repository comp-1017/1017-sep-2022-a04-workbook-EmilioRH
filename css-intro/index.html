CSS coding styles

Browser Defaults you have not supplied any css code
Inline this is for single specific element (tag) instance, the element contains a parameter called "style" style="rule; [rule; ...]" style="color:red; width:50px;"
Embedded coded in your head area of the web page, is started with the style element and contains the selectors and rules for the desired css, applies only to the physical page where the code is placed. Affects all selectors on the page. <style>
selector (rule; [rule; ...])
[selector (rule; [rule; ...]) ...]
<style>		 <style>
selector (rule; [rule; ...])
[selector (rule 400px; [rule color:blue; ...]) ...]
<style>
Stylesheet Similar to embedded, however the selectors and rules are coded in a separate physical fil, this file can be shared throughout the entire website, thus all css is in one common place. selector (rule; [rule; ...])
